Cursor-MCP-ж•°жҚ®еә“

Cursor зҡ„ MCPпјҲModel Context ProtocolпјүеҠҹиғҪе…Ғи®ёејҖеҸ‘иҖ…йҖҡиҝҮй…ҚзҪ® MCP жңҚеҠЎеҷЁжү©еұ• AI е·Ҙе…·зҡ„иғҪеҠӣпјҢд»ҘдёӢжҳҜиҜҰз»ҶдҪҝз”ЁжҢҮеҚ—пјҡ

дёҖгҖҒеҹәзЎҖй…ҚзҪ®жӯҘйӘӨ

  1. вҖӢвҖӢе®үиЈ…дҫқиө–зҺҜеўғвҖӢвҖӢ

    • зЎ®дҝқе·Іе®үиЈ… Node.jsпјҲv18+пјүе’Ң GitгҖӮ
    • еҚҮзә§ Cursor иҮіжңҖж–°зүҲжң¬пјҲEarly Access зүҲж”ҜжҢҒе®Ңж•ҙ MCP еҠҹиғҪпјүгҖӮ

  2. вҖӢвҖӢж·»еҠ  MCP жңҚеҠЎеҷЁвҖӢвҖӢ

    • иҝӣе…Ҙ Cursor и®ҫзҪ® вҶ’ Features вҶ’ MCPпјҢзӮ№еҮ» вҖӢвҖӢ"+ Add New MCP Server"вҖӢвҖӢгҖӮ
    • йҖүжӢ©дј иҫ“зұ»еһӢпјҡ
      • stdioпјҡжң¬ең°е‘Ҫд»ӨиЎҢе·Ҙе…·пјҲеҰӮж–Үд»¶ж“ҚдҪңпјү
      • sse/WebSocketпјҡе®һж—¶ж•°жҚ®жңҚеҠЎпјҲеҰӮзҪ‘з»ңжҗңзҙўпјүгҖӮ

  3. вҖӢвҖӢй…ҚзҪ®ж–Үд»¶зӨәдҫӢвҖӢвҖӢ
    еңЁйЎ№зӣ®ж №зӣ®еҪ•еҲӣе»ә .cursor/mcp.jsonпјҢеҸӮиҖғд»ҘдёӢж јејҸпјҡ

    { "mcpServers": { "web-search": { "command": "npx", "args": ["-y", "@smithery/cli@latest", "run", "@mzxrai/mcp-webresearch"] } } }

дәҢгҖҒзғӯй—Ё MCP е·Ҙе…·й…ҚзҪ®

  1. вҖӢвҖӢWeb жҗңзҙўе·Ҙе…·вҖӢвҖӢ

    • е‘Ҫд»Өпјҡnpx -y @smithery/cli@latest run @mzxrai/mcp-webresearchгҖӮ
    • ж”ҜжҢҒе…ій”®иҜҚжҗңзҙўпјҲеҰӮ cursor mcp serverпјүпјҢиҝ”еӣһе®һж—¶з»“жһңгҖӮ

  2. вҖӢвҖӢBrave жҗңзҙўпјҲйҡҗз§ҒдҝқжҠӨпјүвҖӢвҖӢ

    • йңҖжіЁеҶҢ Brave Search API иҺ·еҸ–еҜҶй’Ҙпјҡ

    "args": ["run", "@smithery-ai/brave-search", "--config", "{\"braveApiKey\":\"YOUR_KEY\"}"]

  3. вҖӢвҖӢGitHub ж“ҚдҪңвҖӢвҖӢ

    • й…ҚзҪ® GitHub MCP жңҚеҠЎеҷЁйңҖдёӘдәәи®ҝй—®д»ӨзүҢпјҲPATпјүпјҡ

    "env": { "GITHUB_PERSONAL_ACCESS_TOKEN": "your_token" }

дёүгҖҒдҪҝз”ЁжҠҖе·§

  • вҖӢвҖӢи°ғз”Ёж–№ејҸвҖӢвҖӢпјҡеңЁ Cursor зҡ„иҒҠеӨ©жЎҶпјҲе·ҰдёӢи§’ "Edit" йқўжқҝпјүиҫ“е…ҘжҢҮд»ӨпјҢеҰӮпјҡ
    @agent Search latest news about MCPгҖӮ
  • вҖӢвҖӢи°ғиҜ•е»әи®®вҖӢвҖӢпјҡиӢҘжңҚеҠЎеҷЁжңӘе“Қеә”пјҢжЈҖжҹҘ Node.js зҺҜеўғжҲ–йҖҡиҝҮ Cursor AI иҫ“е…Ҙ её®жҲ‘жҺ’жҹҘ MCP иҝһжҺҘй—®йўҳгҖӮ

еӣӣгҖҒжіЁж„ҸдәӢйЎ№

  • MCP е·Ҙе…·д»…йҷҗ вҖӢвҖӢComposer AgentвҖӢвҖӢ дҪҝз”ЁпјҲж–°зүҲ Cursor дёӯжӣҙеҗҚдёә "Edit" йқўжқҝпјүгҖӮ
  • Windows з”ЁжҲ·йңҖзЎ®дҝқе‘Ҫд»ӨиЎҢе·Ҙе…·пјҲеҰӮ PowerShellпјүжңүжү§иЎҢжқғйҷҗгҖӮ
