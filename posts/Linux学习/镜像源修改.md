你想用 `vim` 编辑 Ubuntu 的镜像源文件，以下是详细操作步骤：

---

## 使用 vim 修改 Ubuntu 镜像源 `/etc/apt/sources.list`

### 1. 打开终端，输入命令打开文件

```bash
sudo vim /etc/apt/sources.list
```

> `sudo` 是因为该文件需要管理员权限修改。

---

### 2. 进入 `vim` 后的基本操作

- **进入编辑模式**：按 `i` 键进入插入模式
    
- **删除全部内容**：
    
    - 按 `Esc` 键退出插入模式
        
    - 输入命令 `:%d`（删除整个文件内容），然后按 `Enter`
        
- **粘贴内容**：进入插入模式（按 `i`），然后右键粘贴或者用鼠标中键粘贴镜像源内容（Linux 终端支持）
    
- **手动输入内容** 也可以在插入模式直接敲入
    

---

### 3. 复制以下多源内容粘贴进去（清华+阿里+网易示例）

```plaintext
# 清华大学镜像源
deb https://mirrors.tuna.tsinghua.edu.cn/ubuntu/ jammy main restricted universe multiverse
deb https://mirrors.tuna.tsinghua.edu.cn/ubuntu/ jammy-updates main restricted universe multiverse
deb https://mirrors.tuna.tsinghua.edu.cn/ubuntu/ jammy-backports main restricted universe multiverse
deb https://mirrors.tuna.tsinghua.edu.cn/ubuntu/ jammy-security main restricted universe multiverse

# 阿里云镜像源
deb http://mirrors.aliyun.com/ubuntu/ jammy main restricted universe multiverse
deb http://mirrors.aliyun.com/ubuntu/ jammy-updates main restricted universe multiverse
deb http://mirrors.aliyun.com/ubuntu/ jammy-backports main restricted universe multiverse
deb http://mirrors.aliyun.com/ubuntu/ jammy-security main restricted universe multiverse

# 网易镜像源
deb http://mirrors.163.com/ubuntu/ jammy main restricted universe multiverse
deb http://mirrors.163.com/ubuntu/ jammy-updates main restricted universe multiverse
deb http://mirrors.163.com/ubuntu/ jammy-backports main restricted universe multiverse
deb http://mirrors.163.com/ubuntu/ jammy-security main restricted universe multiverse
```

---

### 4. 保存并退出 `vim`

- 先按 `Esc` 退出插入模式
    
- 输入 `:wq` （write + quit），然后按 `Enter` 保存并退出
    

---

### 5. 更新软件包缓存

```bash
sudo apt update
```
