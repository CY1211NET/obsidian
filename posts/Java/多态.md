## ✅ 一、子类继承抽象类可以做什么？

|子类行为|是否允许|说明|
|---|---|---|
|实现抽象方法|✅|必须实现（否则自身也必须是抽象类）|
|添加新属性|✅|子类可以有自己独有的属性|
|添加新方法|✅|子类可扩展额外行为|
|重写父类方法|✅||

非常好，下面我们详细讲解 Java 中的核心面向对象特性之一：**多态（Polymorphism）**，并结合“抽象类 + 子类 + 多态”一起理解。

---

# ✅ Java 多态详解（含抽象类与子类示例）

---

## 一、什么是多态？

**多态（Polymorphism）**指的是：

> 同一个引用类型的变量，**在不同情况下执行不同的行为**，**运行时根据对象的真实类型决定方法的执行效果**。

简单来说：

> **父类引用指向子类对象，调用的方法由子类决定。**

---

## 二、实现多态的三要素（条件）

|条件|示例|说明|
|---|---|---|
|有继承关系|`class Manager extends Employee`|必须是父子类|
|方法重写（Override）|子类重写父类的 `work()` 方法|方法签名一致、内容不同|
|父类引用指向子类对象|`Employee e = new Manager()`|通过向上转型实现|

---

## 三、代码示例：抽象类 + 子类 + 多态

### 抽象父类 Employee：

```java
public abstract class Employee {
    String name;

    public Employee(String name) {
        this.name = name;
    }

    public abstract void work(); // 抽象方法
}
```

### 子类 Manager：

```java
public class Manager extends Employee {
    double bonus;

    public Manager(String name, double bonus) {
        super(name);
        this.bonus = bonus;
    }

    @Override
    public void work() {
        System.out.println(name + " 正在管理团队，奖金：" + bonus);
    }
}
```

### 子类 Developer：

```java
public class Developer extends Employee {
    String language;

    public Developer(String name, String language) {
        super(name);
        this.language = language;
    }

    @Override
    public void work() {
        System.out.println(name + " 使用 " + language + " 编写代码。");
    }
}
```

### 测试类（多态演示）：

```java
public class Test {
    public static void main(String[] args) {
        Employee e1 = new Manager("张经理", 8000);
        Employee e2 = new Developer("李程序员", "Java");

        e1.work(); // 调用 Manager 的 work
        e2.work(); // 调用 Developer 的 work
    }
}
```

---

## 四、输出结果

```
张经理 正在管理团队，奖金：8000.0
李程序员 使用 Java 编写代码。
```

---

## 五、多态的好处

|优点|说明|
|---|---|
|接口统一|用父类或接口类型统一接收不同子类|
|扩展性强|新增子类无需修改原有逻辑|
|代码更灵活|方法调用由运行时动态决定，解耦合|

---

## 六、多态的局限性（只能访问父类成员）

```java
Employee e = new Developer("李程序员", "Java");
e.work(); // ✅
e.language; // ❌ 编译错误，因为父类中没有 language 属性
```

如果需要访问子类独有成员，可以强制向下转型：

```java
Developer dev = (Developer) e;
System.out.println(dev.language); // Java
```
