# 📌 位运算符分类与讲解

在 Java 里，**位运算是直接对二进制位进行操作**，运算速度快，经常用于算法优化、权限控制、状态压缩等场景。

---

## 1️⃣ 按位与 `&`

- **规则**：两个二进制位同时为 `1`，结果才为 `1`。
    
- **示例**：
    

```java
int a = 6;   // 二进制 110
int b = 3;   // 二进制 011
System.out.println(a & b); // 结果 2（二进制 010）
```

---

## 2️⃣ 按位或 `|`

- **规则**：只要有一个为 `1`，结果就是 `1`。
    
- **示例**：
    

```java
int a = 6;   // 110
int b = 3;   // 011
System.out.println(a | b); // 结果 7（二进制 111）
```

---

## 3️⃣ 按位异或 `^`

- **规则**：两个位相同为 `0`，不同为 `1`。
    
- **示例**：
    

```java
int a = 6;   // 110
int b = 3;   // 011
System.out.println(a ^ b); // 结果 5（二进制 101）
```

⚡ 常用技巧：

- `x ^ x = 0`
    
- `x ^ 0 = x`
    
- 交换两数不用临时变量：
    

```java
int a = 3, b = 5;
a = a ^ b;
b = a ^ b; // (a ^ b) ^ b = a
a = a ^ b; // (a ^ b) ^ a = b
```

---

## 4️⃣ 按位取反 `~`

- **规则**：0 变 1，1 变 0。
    
- **示例**：
    

```java
int a = 6; // 0000 0110
System.out.println(~a); // 结果 -7（二进制补码 1111 1001）
```

⚠️ Java 中整数是 **补码存储**，所以取反后会得到负数。

---

## 5️⃣ 左移 `<<`

- **规则**：所有位向左移动，右边补 `0`。
    
- **效果**：相当于乘以 `2^n`。
    
- **示例**：
    

```java
int a = 3; // 0011
System.out.println(a << 2); // 结果 12（二进制 1100）
```

---

## 6️⃣ 右移 `>>`

- **规则**：所有位向右移动，左边补符号位（正数补 `0`，负数补 `1`）。
    
- **效果**：相当于除以 `2^n`（向下取整）。
    
- **示例**：
    

```java
int a = 12; // 1100
System.out.println(a >> 2); // 结果 3（二进制 0011）
```

---

## 7️⃣ 无符号右移 `>>>`

- **规则**：所有位向右移动，左边补 `0`（不管正负）。
    
- **示例**：
    

```java
int a = -12;
System.out.println(a >> 2);  // 结果 -3（补符号位）
System.out.println(a >>> 2); // 很大的正数（因为左边补 0）
```

# 📌 位运算实际应用

1. **快速判断奇偶**
    

```java
if ((n & 1) == 0) {
    System.out.println("偶数");
} else {
    System.out.println("奇数");
}
```

2. **清零某一位**
    

```java
int n = 13;  // 1101
n = n & (~(1 << 2)); // 清除第 2 位（从右数第0位开始）
System.out.println(n); // 结果 9（二进制 1001）
```

3. **状态压缩（存储多个开关状态）**
    

```java
int status = 0;
status |= (1 << 2); // 打开第 2 个开关
status |= (1 << 5); // 打开第 5 个开关

System.out.println((status & (1 << 2)) != 0); // 检查第 2 个开关是否开
```

4. **异或求只出现一次的数**（LeetCode 136）
    

```java
int result = 0;
for (int num : nums) {
    result ^= num;
}
return result;
```

✅ 总结口诀：

- `&` 都 1 才 1
    
- `|` 有 1 就 1
    
- `^` 不同才 1
    
- `~` 取反
    
- `<<` 左移乘 2
    
- `>>` 右移除 2（保符号）
    
- `>>>` 右移补 0（无符号）
