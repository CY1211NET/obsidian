# ✅ Java 面向对象核心：抽象类、父类与子类总结

---

## 一、父类（基类）

**概念：**  
父类是被继承的类，包含了通用的属性和方法。子类通过 `extends` 继承它。

### 父类特点：

|特点|说明|
|---|---|
|包含公共属性和方法|如：`name`、`salary`|
|可以被继承|子类使用 `extends 父类` 继承|
|构造方法不能被继承|但可以通过 `super()` 调用|
|可以被向上转型|`父类变量 = new 子类对象()`|

### 示例：

```java
public class Employee {
    String name;
    double salary;

    public Employee(String name, double salary) {
        this.name = name;
        this.salary = salary;
    }

    public void work() {
        System.out.println(name + " 正在工作。");
    }
}
```

---

## 二、子类（派生类）

**概念：**  
子类继承父类，可以使用父类的非私有成员，也可以扩展或修改它的行为。

### 子类特点：

|特点|说明|
|---|---|
|使用 `extends` 关键字继承父类||
|继承父类的**非私有成员**（变量和方法）||
|可新增属性、方法||
|可**重写**（Override）父类方法||
|构造器中必须先调用父类构造器（`super(...)`）||

### 示例：

```java
public class Manager extends Employee {
    double bonus;

    public Manager(String name, double salary, double bonus) {
        super(name, salary); // 调用父类构造器
        this.bonus = bonus;
    }

    @Override
    public void work() {
        super.work(); // 调用父类的 work 方法
        System.out.println(name + " 正在管理项目。");
    }
}
```

## 三、抽象类（`abstract class`）

**概念：**  
抽象类是一种**不能实例化**的特殊类，定义一些“必须由子类实现”的行为（抽象方法）。

### 抽象类特点：

|特点|说明|
|---|---|
|不能实例化|不能 `new` 抽象类|
|可以有构造方法、普通方法||
|至少包含一个抽象方法|使用 `abstract` 关键字定义|
|子类**必须**实现抽象方法|否则子类也必须是抽象类|

### 示例：

```java
public abstract class Employee {
    String name;
    double salary;

    public Employee(String name, double salary) {
        this.name = name;
        this.salary = salary;
    }

    public abstract void work(); // 抽象方法
}
```

---

## 四、三者的继承关系与使用对比

|项目|父类|子类|抽象类|
|---|---|---|---|
|是否能实例化|✅|✅|❌|
|是否可继承|✅|✅|✅|
|是否可重写方法|❌（非 final 方法可）|✅|✅|
|是否可以定义抽象方法|❌|❌|✅|
|构造器是否可继承|❌|❌|❌（但可调用）|
|是否支持多态|✅|✅|✅|

## 五、举例说明

```java
abstract class Employee {
    String name;
    Employee(String name) { this.name = name; }
    abstract void work(); // 抽象方法
}

class Manager extends Employee {
    Manager(String name) { 
    super(name); 
    }
    void work() {
        System.out.println(name + " 管理团队。");
    }
}

class Developer extends Employee {
    Developer(String name) { super(name); }
    void work() {
        System.out.println(name + " 编写代码。");
    }
}
```

**多态调用：**

```java
Employee e1 = new Manager("张经理");
Employee e2 = new Developer("李程序员");

e1.work(); // 输出：张经理 管理团队。
e2.work(); // 输出：李程序员 编写代码。
```

## 六、关键词速记口诀：

```
父类通用模板用，子类继承加扩展；
抽象不能被实例，方法空壳子类填；
super 调父构或父法，this 指向自身源；
多态调用靠重写，运行多变编译稳。
```
