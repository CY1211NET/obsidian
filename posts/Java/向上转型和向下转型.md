# Java 多态与转型思维导图结构

```
Java 多态与转型
│
├── 抽象类与继承
│     ├── 抽象类：不能实例化，含抽象方法
│     ├── 子类继承：实现抽象方法，可增加属性
│
├── 多态（Polymorphism）
│     ├── 父类引用指向子类对象
│     ├── 动态绑定：运行时调用子类方法
│
├── 向上转型（Upcasting）
│     ├── 父类变量引用子类对象
│     ├── 自动类型转换
│     ├── 只能访问父类方法和属性
│     ├── 多态基础
│
├── 向下转型（Downcasting）
│     ├── 父类引用强制转为子类类型
│     ├── 访问子类特有成员
│     ├── 需确保实际对象是子类类型
│     ├── 常用 instanceof 做安全判断
│
└── 设计模式中的应用
      ├── 多态实现灵活扩展
      ├── 向上转型用于统一接口编程
      ├── 向下转型访问子类特有功能
```

# 简单示例代码

```java
// 抽象类
abstract class Employee {
    String name;
    public Employee(String name) { this.name = name; }
    public abstract void work();
}

// 子类
class Manager extends Employee {
    double bonus;
    public Manager(String name, double bonus) {
        super(name);
        this.bonus = bonus;
    }
    @Override
    public void work() {
        System.out.println(name + " 管理团队，奖金：" + bonus);
    }
    public void manage() {
        System.out.println(name + " 正在做管理工作。");
    }
}

public class Test {
    public static void main(String[] args) {
        // 向上转型，多态体现
        Employee e = new Manager("李经理", 5000);
        e.work();

        // 向下转型，访问子类特有方法
        if (e instanceof Manager) {
            Manager m = (Manager) e;
            m.manage();
            System.out.println("奖金：" + m.bonus);
        }
    }
}
```
