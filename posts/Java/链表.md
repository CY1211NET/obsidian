链表（**Linked List**）是一个非常重要的数据结构，它通过一系列 **节点**（Node）连接在一起，节点中包含数据和指向下一个节点的指针。链表的主要特点是 **动态分配内存**，可以根据需要灵活地增加和删除节点。因此，链表在一些需要频繁插入和删除操作的场景中比数组更加高效。

## ✅ 一、链表的基本概念

### 1. **链表的基本结构**

每个链表由多个 **节点**（Node）组成，每个节点通常包含两个部分：

- **数据部分**：存储节点的数据，可以是任何类型的数据。
    
- **指针部分**：指向链表中的下一个节点。
    

链表的最初节点叫做 **头节点**（Head），链表的最后一个节点的指针指向 **null**（单链表）或指向头节点（循环链表）。

### 2. **链表的类型**

链表有多种实现方式，常见的有以下几种：

- **单链表（Singly Linked List）**：每个节点只包含指向下一个节点的指针。
    
- **双链表（Doubly Linked List）**：每个节点包含指向前一个节点和下一个节点的指针。
    
- **循环链表（Circular Linked List）**：链表的最后一个节点指向头节点，形成循环。

## ✅ 二、链表的基本操作

### 1. **单链表的基本操作**

单链表的基本操作包括插入、删除、遍历等。

#### 单链表节点的定义

```java
// 定义单链表节点
class ListNode {
    int val;
    ListNode next;

    ListNode(int val) {
        this.val = val;
        this.next = null;
    }
}
```

#### 单链表的常见操作

```java
public class SinglyLinkedList {
    private ListNode head;

    // 插入节点到链表头部
    public void insertAtHead(int value) {
        ListNode newNode = new ListNode(value);
        newNode.next = head;
        head = newNode;
    }

    // 删除链表头部的节点
    public void deleteAtHead() {
        if (head != null) {
            head = head.next;
        }
    }

    // 遍历链表并打印
    public void printList() {
        ListNode current = head;
        while (current != null) {
            System.out.print(current.val + " -> ");
            current = current.next;
        }
        System.out.println("null");
    }

    public static void main(String[] args) {
        SinglyLinkedList list = new SinglyLinkedList();
        list.insertAtHead(10);
        list.insertAtHead(20);
        list.insertAtHead(30);
        list.printList(); // 30 -> 20 -> 10 -> null
        list.deleteAtHead();
        list.printList(); // 20 -> 10 -> null
    }
}
```

### 2. **双链表的基本操作**

在 **双链表** 中，每个节点不仅有指向下一个节点的指针 `next`，还包含指向前一个节点的指针 `prev`。这使得双链表可以在两端进行遍历。

#### 双链表节点的定义

```java
// 定义双链表节点
class DoublyListNode {
    int val;
    DoublyListNode next;
    DoublyListNode prev;

    DoublyListNode(int val) {
        this.val = val;
        this.next = null;
        this.prev = null;
    }
}
```

#### 双链表的常见操作

```java
public class DoublyLinkedList {
    private DoublyListNode head;

    // 插入节点到链表尾部
    public void insertAtTail(int value) {
        DoublyListNode newNode = new DoublyListNode(value);
        if (head == null) {
            head = newNode;
        } else {
            DoublyListNode current = head;
            while (current.next != null) {
                current = current.next;
            }
            current.next = newNode;
            newNode.prev = current;
        }
    }

    // 删除指定节点
    public void delete(int value) {
        DoublyListNode current = head;
        while (current != null) {
            if (current.val == value) {
                if (current.prev != null) {
                    current.prev.next = current.next;
                }
                if (current.next != null) {
                    current.next.prev = current.prev;
                }
                if (current == head) {
                    head = current.next;
                }
                break;
            }
            current = current.next;
        }
    }

    // 遍历链表并打印
    public void printList() {
        DoublyListNode current = head;
        while (current != null) {
            System.out.print(current.val + " <-> ");
            current = current.next;
        }
        System.out.println("null");
    }

    public static void main(String[] args) {
        DoublyLinkedList list = new DoublyLinkedList();
        list.insertAtTail(10);
        list.insertAtTail(20);
        list.insertAtTail(30);
        list.printList(); // 10 <-> 20 <-> 30 <-> null
        list.delete(20);
        list.printList(); // 10 <-> 30 <-> null
    }
}
```

### 3. **循环链表**

在 **循环链表** 中，最后一个节点的 `next` 指向头节点，形成循环。因此，循环链表没有尾节点和头节点的区别，适用于需要循环遍历的场景。

#### 循环链表节点的定义

```java
// 定义循环链表节点
class CircularListNode {
    int val;
    CircularListNode next;

    CircularListNode(int val) {
        this.val = val;
        this.next = null;
    }
}
```

#### 循环链表的常见操作

```java
public class CircularLinkedList {
    private CircularListNode head;

    // 插入节点到链表尾部
    public void insertAtTail(int value) {
        CircularListNode newNode = new CircularListNode(value);
        if (head == null) {
            head = newNode;
            newNode.next = head; // 循环连接
        } else {
            CircularListNode current = head;
            while (current.next != head) {
                current = current.next;
            }
            current.next = newNode;
            newNode.next = head; // 循环连接
        }
    }

    // 遍历链表
    public void printList() {
        if (head == null) return;

        CircularListNode current = head;
        do {
            System.out.print(current.val + " -> ");
            current = current.next;
        } while (current != head);
        System.out.println("back to head");
    }

    public static void main(String[] args) {
        CircularLinkedList list = new CircularLinkedList();
        list.insertAtTail(10);
        list.insertAtTail(20);
        list.insertAtTail(30);
        list.printList(); // 10 -> 20 -> 30 -> back to head
    }
}
```

## ✅ 三、链表的常见问题与练习题

### **练习题 1：反转单链表**

**问题描述**：  
给定一个单链表，反转链表并返回反转后的头节点。

**解法思路**：  
遍历链表，逐个反转节点的指针。

```java
public class ReverseLinkedList {
    public ListNode reverse(ListNode head) {
        ListNode prev = null;
        ListNode current = head;
        while (current != null) {
            ListNode next = current.next;
            current.next = prev;
            prev = current;
            current = next;
        }
        return prev;
    }

    public static void main(String[] args) {
        SinglyLinkedList list = new SinglyLinkedList();
        list.insertAtHead(10);
        list.insertAtHead(20);
        list.insertAtHead(30);
        list.printList(); // 30 -> 20 -> 10 -> null
        list.head = new ReverseLinkedList().reverse(list.head);
        list.printList(); // 10 -> 20 -> 30 -> null
    }
}
```

### **练习题 2：查找链表的中间节点**

**问题描述**：  
给定一个单链表，找出链表的中间节点。如果链表有两个中间节点，则返回第二个中间节点。

**解法思路**：  
使用 **快慢指针**，慢指针每次走一步，快指针每次走两步，最终当快指针到达链表末尾时，慢指针所在的位置即为链表的中间节点。

```java
public class MiddleOfLinkedList {
    public ListNode middleNode(ListNode head) {
        ListNode slow = head;
        ListNode fast = head;
        while (fast != null && fast.next != null) {
            slow = slow.next;
            fast = fast.next.next;
        }
        return slow;
    }

    public static void main(String[] args) {
        SinglyLinkedList list = new SinglyLinkedList();  
             list.insertAtHead(10);  
             list.insertAtHead(20);  
             list.insertAtHead(30);  
             list.insertAtHead(40);  
             list.insertAtHead(50);  
ListNode middle = new MiddleOfLinkedList().middleNode(list.head);  
System.out.println(middle.val); // 30  
}  
}
```





### **练习题 3：判断链表是否有环**
**问题描述**：
给定一个单链表，判断链表中是否有环。

**解法思路**：
使用 **快慢指针**，如果快指针和慢指针相遇，则链表有环，否则没有。

```java
public class LinkedListCycle {
    public boolean hasCycle(ListNode head) {
        ListNode slow = head;
        ListNode fast = head;
        while (fast != null && fast.next != null) {
            slow = slow.next;
            fast = fast.next.next;
            if (slow == fast) {
                return true;
            }
        }
        return false;
    }

    public static void main(String[] args) {
        SinglyLinkedList list = new SinglyLinkedList();
        list.insertAtHead(10);
        list.insertAtHead(20);
        list.insertAtHead(30);
        // Create a cycle for testing
        list.head.next.next.next = list.head.next;
        LinkedListCycle cycleChecker = new LinkedListCycle();
        System.out.println(cycleChecker.hasCycle(list.head));  // true
    }
}
````

## 1️⃣ 尾部插入节点：`insertAtTail(int data)`

```java
public ListNode insertAtTail(ListNode head, int data) {
    ListNode newNode = new ListNode(data);
    if (head == null) return newNode;
    ListNode curr = head;
    while (curr.next != null) {
        curr = curr.next;
    }
    curr.next = newNode;
    return head;
}
```

## 2️⃣ 删除值为 `key` 的第一个节点：`deleteNode(int key)`

```java
public ListNode deleteNode(ListNode head, int key) {
    if (head == null) return null;
    if (head.val == key) return head.next; // 删除头结点

    ListNode curr = head;
    while (curr.next != null && curr.next.val != key) {
        curr = curr.next;
    }

    if (curr.next != null) {
        curr.next = curr.next.next; // 跳过目标节点
    }

    return head;
}
```

## 3️⃣ 查找链表是否包含某值：`contains(int key)`

```java
public boolean contains(ListNode head, int key) {
    ListNode curr = head;
    while (curr != null) {
        if (curr.val == key) return true;
        curr = curr.next;
    }
    return false;
}
```

## 4️⃣ 链表反转：`reverse()`

```java
public ListNode reverse(ListNode head) {
    ListNode prev = null;
    ListNode curr = head;

    while (curr != null) {
        ListNode next = curr.next; // 暂存下一个节点
        curr.next = prev;          // 当前节点指向前一个
        prev = curr;
        curr = next;
    }

    return prev; // 新的头节点
}
```

## 6️⃣ 合并两个有序链表（LeetCode 21）

```java
public ListNode mergeTwoLists(ListNode l1, ListNode l2) {
    ListNode dummy = new ListNode(-1);
    ListNode curr = dummy;

    while (l1 != null && l2 != null) {
        if (l1.val <= l2.val) {
            curr.next = l1;
            l1 = l1.next;
        } else {
            curr.next = l2;
            l2 = l2.next;
        }
        curr = curr.next;
    }

    curr.next = (l1 != null) ? l1 : l2;
    return dummy.next;
}
```

## 📌 总结

- **链表** 是动态数据结构，插入和删除操作高效，但查找操作较慢。
    
- 常见的链表类型包括 **单链表**、**双链表** 和 **循环链表**。
    
- **单链表**：每个节点只包含一个指向下一个节点的指针。
    
- **双链表**：每个节点有两个指针，分别指向前后节点。
    
- **循环链表**：最后一个节点的指针指向头节点，形成循环。
    
- 练习题帮助理解链表的基本操作，如反转链表、查找中间节点、判断环等。
