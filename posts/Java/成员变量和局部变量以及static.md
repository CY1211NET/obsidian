# Java 变量与 `this` 关键字完整总结

|变量类型 / 关键字|定义位置|作用范围|生命周期|初始化要求|备注与用法说明|
|---|---|---|---|---|---|
|**成员变量**|类中，方法外|类的所有实例方法可访问|对象创建到销毁|自动初始化默认值|每个对象拥有独立副本，存储对象属性|
|**局部变量**|方法、构造器、代码块内|仅限当前方法或代码块|方法调用期间|必须显式初始化|临时存储数据，出了作用域即销毁|
|**静态变量**|类中，方法外，`static`修饰|类所有对象共享|类加载到卸载|自动初始化默认值|类变量，类似全局变量，用于共享数据|
|**`this` 关键字**|实例方法或构造器内使用|指向当前对象|随对象存在|-|用来区分成员变量与局部变量，调用当前对象方法，调用构造器等|

---

## 详细说明

### 1. 成员变量

- 描述对象状态，属于对象的属性。
    
- 访问时如果局部变量和成员变量重名，必须用 `this` 访问成员变量。
    
- 如无显式赋值，自动赋默认值（数字型0，布尔型false，引用null）。
    

### 2. 局部变量

- 定义在方法、构造器或代码块内。
    
- 作用范围仅限于所在代码块或方法。
    
- 必须初始化，且不能被方法外访问。
    

### 3. 静态变量

- 用 `static` 修饰，属于类，不属于对象。
    
- 类所有对象共享同一份数据。
    
- 常用于计数器、配置参数等。
    

### 4. `this` 关键字

- 代表当前对象引用。
    
- 用于区分成员变量和局部变量。
    
- 可以调用当前对象的其他方法或构造器。
    
- 支持链式调用返回当前对象。
    

---

## 通俗示例

```java
class Student {
    String name;  // 成员变量

    public Student(String name) {
        this.name = name;  // this.name 是成员变量，name 是参数
    }

    public void sayName(String name) {
        System.out.println("参数名字：" + name);
        System.out.println("成员变量名字：" + this.name);
    }

    public Student setName(String name) {
        this.name = name;
        return this;  // 支持链式调用
    }

    public void print() {
        System.out.println("学生姓名：" + this.name);
    }
}
```

## 常见面试题简答

|题目|答案简述|
|---|---|
|成员变量和局部变量有什么区别？|成员变量属于对象，生命周期长，自动初始化；局部变量作用域短，必须初始化。|
|Java中有全局变量吗？|没有真正全局变量，`static`变量可做类似全局变量使用。|
|`this`关键字有什么用？|表示当前对象，区分成员变量与局部变量，调用本类方法和构造器等。|
|如何访问被局部变量遮蔽的成员变量？|用 `this.变量名` 访问成员变量。|
